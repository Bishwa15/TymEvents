Tym Events for Google Calendar and Calendar apps

List of Tym events for Calendar apps

Plenty of online Nepali Calendars are available presently but there is a gap between making those events available to the popular calendar applications like Google Calendar and Calendar app on mac. Google Calendar and Calendar on mac have a nice way to sync the 3rd party calendar events. This project is to provide a Internet Calendaring api for major events in Nepal.

This project aims to maintain a calendar events of Nepal that can be imported to popular calendar apps such as Google Calendar and Calendar app on Mac. The Events provided through this api is based on data from HamroPatro

Installation

Subscribe to Tym events on Calendar app on Mac

  1. Go to File > New Calendar Subscription
  2. Enter http://shresthasushil.com.np/NepaliEvents/Nepali%20Events.ics on Calendar URL.
  3. Click subscribe.
  4. Provide a name for the subscription and press Ok

Subscribe to Tym events on Google Calendar

  1. Go to Add Calendar > From URL
  2. Enterhttp://shresthasushil.com.np/NepaliEvents/Nepali%20Events.ics on Calendar URL.
  3. Click Add Calendar.

Calendars

  • Tym Dates api available through http://shresthasushil.com.np/NepaliEvents/Nepali%20Dates.ics
  • Tym Events api available through http://shresthasushil.com.np/NepaliEvents/Nepali%20Events.ics
Also found a Google maintained calendar for Tym events. If you prefer less verbose calendar events check the calendar events from Google. https://calendar.google.com/calendar/ical/en.np%23holiday%40group.v.calendar.google.com/public/basic.ics
